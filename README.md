# Brick Breaker

A modern neon-themed **Brick Breaker** web game built using **HTML, CSS, and JavaScript**. This project recreates the classic arcade experience with a responsive design, smooth animations, and mobile-friendly controls.

# Features

- Modern neon-themed UI
- Responsive design for desktop and mobile
- Mobile touch controls
- Smooth paddle movement
- Dynamic ball physics
- Multiple brick layouts
- Lives system
- Score tracking
- High score tracking
- Level progression
- Smooth animations
- Glassmorphism inspired interface
- Lightweight and fast

---

# Mobile Support

Unlike many browser-based Brick Breaker games that rely only on keyboard controls, this project is optimized for mobile devices.

Simply tap anywhere on the screen and the paddle smoothly slides to the selected position, creating an intuitive touch-based gameplay experience.

# Gameplay

1. Launch the game.
2. Control the paddle.
3. Bounce the ball.
4. Break every brick.
5. Clear the level.
6. Beat your high score.

---

# Controls

## Desktop

- Move mouse to control paddle

## Mobile

- Tap anywhere on the screen
- Paddle smoothly slides to the tapped position
